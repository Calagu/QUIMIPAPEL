# QumiPedidos - Semana 1

Proyecto inicial para IntelliJ + Scene Builder.

Esta versión solo incluye la preparación del entorno y bocetos de pantallas FXML. Todavía no contiene toda la lógica de la aplicación.

## Abrir en IntelliJ

1. File > Open
2. Selecciona esta carpeta
3. Espera a que Maven cargue las dependencias
4. Ejecuta:

```bash
mvn javafx:run
```

## Abrir bocetos en Scene Builder

Los FXML están en:

```text
src/main/resources/com/qumi/semana1/view/
```

## Base de datos

Importa el archivo:

```text
02_Modelo_Datos_Inicial_HeidiSQL.sql
```

en HeidiSQL.

Base de datos:

```sql
qumi_pedidos_semana1
```
